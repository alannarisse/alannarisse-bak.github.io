CSS Display property: inline VS display: inline-block VS display: block.

Fun with HTML5 and CSS3

  • Mathew Lein - sad trombone and more!
- inline element has no line break before or after it, and it tolerates HTML elements next to it. - A block element has some whitespace above and below it and does not tolerate any HTML elements next to it. - An inline-block element is placed as an inline element (on the same line as adjacent content), but it behaves as a block element.
